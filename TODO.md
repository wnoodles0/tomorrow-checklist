# เช็กลิสต์งาน (TODO Checklist)

## ระยะที่ 1: โครงสร้างพื้นฐาน (Foundation)

- [ ] สร้างโครงสร้างไฟล์ HTML พื้นฐานใน `index.html`
- [ ] เชื่อมโยงไฟล์ `styles/main.css` และ `scripts/app.js`
- [ ] ตั้งค่าสไตล์พื้นฐานใน `main.css` (เช่น font, background color, เน้นการออกแบบที่อ่านง่ายบนมือถือ)
- [ ] **สร้างไฟล์ `data/items.json`** (เสร็จสิ้นแล้ว)

## ระยะที่ 2: ฟังก์ชันหลัก (Core Functionality)

- [ ] **โหลดข้อมูล:** อ่านรายการสินค้าจาก `data/items.json` และแสดงผลเรียงตาม `order`
- [ ] **บันทึก/โหลดสถานะ:** โหลดสถานะ `done` จาก Local Storage (ถ้ามี) และบันทึกกลับทันทีเมื่อมีการเปลี่ยนแปลง
- [ ] **การแสดงผล:** แสดงรายการพร้อม Checkbox สำหรับติ๊กเช็ก
- [ ] **การติ๊กเช็ก:** สร้างฟังก์ชันสำหรับทำเครื่องหมายว่าเสร็จสิ้น (Toggle Complete) โดยที่รายการที่ติ๊กแล้วจะจางลงแต่ไม่หายไป

## ระยะที่ 3: ฟีเจอร์ตามแนวทาง "รายการแบบมีลำดับคงที่"

- [ ] **ช่องค้นหาเร็ว:** เพิ่มช่องค้นหาเพื่อกรองแสดงเฉพาะรายการที่ต้องการ
- [ ] **UX การแสดงผล:**
    - [ ] เน้นการออกแบบที่สแกนง่าย ตัวหนังสือใหญ่พอ
    - [ ] **เลื่อนอัตโนมัติ:** เลื่อนหน้าจอไปยังรายการถัดไปที่ยังไม่ติ๊ก (Optional)
    - [ ] **โหมดเต็มหน้าจอ:** เพิ่มปุ่มหรือฟังก์ชันสำหรับโหมดเต็มหน้าจอ (Full-screen checklist)
- [ ] **ฟีเจอร์ย่อย:**
    - [ ] ปุ่ม "รีเซ็ตเช็กลิสต์" (ล้างเครื่องหมายถูกทั้งหมด แต่คงลำดับไว้)
    - [ ] ปุ่มบันทึกลิสต์เป็นไฟล์ (Export JSON/CSV) (Optional)
