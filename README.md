# Tomorrow Checklist

## เป้าหมาย (Goal)

แอปพลิเคชันรายการตรวจสอบ (Checklist) ที่เรียบง่ายและใช้งานง่ายบนมือถือ สำหรับการวางแผนงานและสิ่งที่ต้องทำสำหรับวันถัดไป (Tomorrow's tasks) เพื่อช่วยให้ผู้ใช้เริ่มต้นวันใหม่ได้อย่างมีประสิทธิภาพ

## User Stories

ในฐานะผู้ใช้ ฉันต้องการ:

1.  **เพิ่มงานใหม่:** สามารถเพิ่มรายการงานใหม่ลงในเช็กลิสต์ได้
2.  **ทำเครื่องหมายว่าเสร็จสิ้น:** สามารถทำเครื่องหมายงานว่าเสร็จสิ้นแล้ว (Check/Uncheck) เพื่อติดตามความคืบหน้า
3.  **ลบงาน:** สามารถลบรายการงานที่ไม่เกี่ยวข้องออกไปได้
4.  **บันทึกอัตโนมัติ:** รายการงานถูกบันทึกโดยอัตโนมัติ (เช่น ใน Local Storage) เพื่อไม่ให้ข้อมูลสูญหาย
5.  **การออกแบบที่ตอบสนอง:** มีการออกแบบที่ใช้งานง่ายและสวยงามบนอุปกรณ์มือถือ (Mobile-friendly)
